Chicken Danger

Website

Created: 2016

Chicken Danger is a board game that is currently in production and due for a Kickstarter in the near future.


I loved working on this project as the art assets and direction of the game are very low key as it is a family made effort. Pending on the success of the Kickstarter, there will be a page refresh with some higher end assets.


The CSS is designed to give a great presentation and makes use of icon fonts, curved edges, and even some hidden animations.


This project also has several app video games that are almost ready to go as well. They will not be released until the board game is out however, and I am not at liberty to share those. The games were written in Pixi, ES2015, and backbone/backbone.marionette. Their back end components were all written in node, with some PHP tie ins.


Chicken Danger will also launch the GOEcorp platform which hosts a currency system which allows people to buy and earn credits. This also utilizes the chatter, which is used to allow people to chat while playing video games or utilizing any of the platforms web apps.

Crystal Fault

Website

Created: 2014

Crystal Fault was conceptual game that I helped with. I created this page based on very loose requirements handed to me from the game director at the time. The text was mainly used as a placeholder as was much of the code.


The page uses a faux Jquery framework and was written inside of a backbone.marionette application. The presented quite a few challenges when trying to create styling that was global and thematic, but not overly aggressive.


As far as the CSS goes, there are not a lot of different techniques or libraries here. Rather the CSS is all hand written and was tested in all major browsers. This was intentional as we were planning on going with either Sass or less.


The project has been archived and put in remission for now.
